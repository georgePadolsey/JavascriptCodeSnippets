Module: aReplace

aReplace

(require("aReplace"))(properties) → {Boolean}

Replaces all anchor tags (a tags) with a fake address when hovered over; Though it still takes the person to the required place; It degrades to the normal hover over link if javascript is off;
Parameters:
Name Type Description
properties Object The properties for the aReplace
Properties
Name Type Description
container Object The container to run aReplace in aReplace#container
recursive Boolean Whether it will go in the child elements of the container and run aReplace on them aReplace#recursive
text String The, "fake", address of the text when hovered over
Source:
  • aReplace.js, line 48
Returns:
if completed sucessfully _mainly a fallback incase an error didn't correctly break the function_
Type
Boolean
Example

aReplace({
		// Run through the container element
		container: document.getElementById('container'), 
		// Recurse through the child elements of it
		recursive: true, 
		// Make the link appear as http://www.georgep.co.uk/
		text: 'http://www.georgep.co.uk/'
});
