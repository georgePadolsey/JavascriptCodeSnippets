Module: aReplace

aReplace

Replaces all anchor tags (a tags) with a fake address when hovered over; Though it still takes the person to the required place; It degrades to the normal hover over link if javascript is off;
Source:
  • aReplace.js, line 22

Example


aReplace({
    // Run through the container element
    container: document.getElementById('container'), 
    // Recurse through the child elements of it
    recursive: true, 
    // Make the link appear as http://www.georgep.co.uk/
    text: 'http://www.georgep.co.uk/'
});
